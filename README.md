# DSA

This repository contains C++ practice files for data structures and algorithms.

## Contents

- C++ source files for loops, conditionals, data types, and more.
- A clean initial commit with `.gitignore` to avoid checking in compiled binaries.

## Notes

- Use `g++` or any C++ compiler to build the `.cpp` files.
- The repository is public and ready for further DSA exercises.

## Example

```bash
# Build a sample source file
g++ loop11.cpp -o loop11.exe
./loop11.exe
```
